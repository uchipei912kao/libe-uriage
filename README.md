## iPhoneでアプリとして使う方法

### 方法A：Macでサーバー起動 → iPhoneで開く（同じWi-Fi）
1. ターミナルでこのフォルダに移動して以下を実行：
   ```
   cd "/Users/uchikawaippei/Downloads/リベ市場売上管理アプリ"
   python3 -m http.server 8000
   ```
2. MacのIPアドレスを確認（システム設定→Wi-Fi→詳細、例: 192.168.1.5）
3. iPhoneのSafariで `http://192.168.1.5:8000` を開く
4. 共有ボタン →「ホーム画面に追加」→ アイコンで起動

### 方法B：ネットに公開（どこでも使える・推奨）
無料ホスティング（Netlify / Vercel / GitHub Pages）にこのフォルダを上げると、
固定URLでどこからでもアクセス可能。次回セットアップする。

## ファイル構成
- index.html : 画面構造
- style.css  : デザイン（青基調）
- app.js     : 動作ロジック
- data.js    : ダミーデータ（次回ここを実データに差し替え）
- manifest.json / sw.js : PWA設定（ホーム画面アプリ化・オフライン対応）
- icons/     : アプリアイコン

## データ差し替えポイント（次回用メモ）
`data.js` の SOLD_ITEMS / STOCK_ITEMS / SUMMARY を
Supabase or Chrome拡張から取得した実データに置き換えれば本番化できる。
在庫の手編集は localStorage('libe_stock') に保存される。
